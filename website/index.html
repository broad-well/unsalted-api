Installation

Installation is as easy as pie. To do so, create a bookmark on your favorite browser. For the URL of the bookmark, insert the following:

javascript:var a=document.createElement("script");a.setAttribute("src", "https://cdn.rawgit.com/michaelpeng2002/unsalted-api/unusable/unsalted_api.js");a.setAttribute("language", "JavaScript");a.setAttribute("type","text/javascript");document.head.appendChild(a);


Usage

To use UnsaltedAPI, first ensure that the script is available as a bookmark. (see Installation.)
Visit the bookmark to inject the script.
Then, open up a JavaScript console in your browser. You can now use UnsaltedAPI in the terminal. For help, see Manual.
